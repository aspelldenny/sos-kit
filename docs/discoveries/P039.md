## Task 8 idiom transfer
- Banner script pattern (lines 22-28): `HEADER_LINE=$(grep -n "^## .*Active sprint" "$BACKLOG" 2>/dev/null | head -1 | cut -d: -f1)` then `if [ -z "$HEADER_LINE" ]; then HEADER_LINE=$(grep -n "^## " "$BACKLOG" 2>/dev/null | head -1 | cut -d: -f1); FALLBACK_USED=1; fi`.
- Pre-commit adaptation: same grep pattern, but then passes `HEADER_LINE` to awk via `-v hdr="$HEADER_LINE"` to extract section content via `NR==hdr{found=1; next} found && /^## /{exit} found{print}`. This is a clean adaptation — the banner uses the line number to display content, the hook uses it to count items. Idiom is consistent: same grep-with-fallback for the header-find step.
